# Psychic Petals Web Platform

> A dedicated frontend reading experience for the open-source novel, [Psychic Petals](https://github.com/RollieGarcia0031/psychic-petals).

This repository contains the Next.js frontend application designed to beautifully display the **Psychic Petals** story. The story itself is a magical realism and slice-of-life novel set in a world with a corporate-regulated hero ecosystem, focusing on the quiet, personal moments of its characters.

## 🌟 Features

- **Modern Web Experience:** Built with [Next.js](https://nextjs.org/) for a fast, responsive, and seamless reading experience.
- **Dynamic Presentation:** Beautifully renders the markdown-based episodes and chapters of the novel.
- **Tailored Design:** Features a rich, aesthetic UI optimized for readability and immersion into the world of Psychic Petals.

## 🚀 Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
